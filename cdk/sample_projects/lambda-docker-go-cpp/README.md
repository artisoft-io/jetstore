# Lambda Docker Go C++ Example

This example demonstrates how to create an AWS Lambda function using Go with a C++ library, deployed using Docker containers and AWS CDK v2.

## Prerequisites

- AWS CLI configured
- Docker installed
- Go 1.21+
- AWS CDK v2
- Node.js (for CDK)
- jq (for testing)

## Quick Start

1. **Setup the project:**

```bash
chmod +x setup.sh
./setup.sh
```
